# 金盲杖网站首版

这个目录里现在同时保留了网页版和新的安卓原生 APK 工程。

## 当前功能

- 首页提供两个大按钮：`寻找物品` 和 `管理物品`
- NFC 贴片可直接打开专属物品页
- 物品页优先播报名称
- 详细信息分为 `用途`、`注意事项`、`特殊备注`
- 寻物页按“已连接过的设备”展示可寻找物品，并使用单个 `连接并寻找` 按钮
- 管理页仅维护名称、简介播报、用途、注意事项、特殊备注
- 简介播报、用途、注意事项、特殊备注都支持文字和音频
- 页面整体保持大字号、高对比度、读屏友好

## 项目结构

- `app/`：Next.js 页面与全局样式
- `components/`：前端交互组件
- `lib/`：默认物品数据
- `package.json`：Node 依赖与启动脚本

## 本地启动

先安装 Node.js，再在项目目录执行：

```bash
npm install
npm run dev
```

然后在浏览器打开：

```text
http://localhost:3000
```

## 页面地址

- 首页：`/`
- 寻找物品：`/find`
- 管理物品：`/manage`
- 添加新物品：`/manage/new`
- 示例物品页：`/item/cold-medicine`

## 当前说明

- 现在的“播放音频”仍支持浏览器语音播报模拟，也支持上传音频文件
- 现在的“寻找物品”页面已经支持：

  - 编辑名称、简介播报、用途、注意事项、特殊备注

  - 把物品数据改成数据库读取
